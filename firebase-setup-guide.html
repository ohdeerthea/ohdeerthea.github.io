Follow these steps to set up Firebase for your commission queue functionality. If you skip this setup, the site will work with demo data.

Step 1: Create Firebase Project

  1. Go to Firebase Console
  2. Click "Add project" or "Create a project"
  3. Enter your project name (e.g., "oc-gallery")
  4. Follow the setup wizard (you can disable Analytics if not needed)

Step 2: Enable Firestore Database

  1. In your Firebase project, click "Firestore Database" in the left sidebar
  2. Click "Create database"
  3. Choose "Start in test mode" for now (you can secure it later)
  4. Select a location close to your users

Step 3: Get Firebase Configuration

  1. In Firebase Console, click the gear icon ‚Üí "Project settings"
  2. Scroll down to "Your apps" section
  3. Click the web app icon "</>" to add a web app
  4. Enter an app nickname (e.g., "OC Gallery Web")
  5. Copy the configuration object that appears

Step 4: Update Configuration File

Open scripts/firebase-config.js and replace the placeholder config with your actual Firebase configuration:

const firebaseConfig = {
  apiKey: "your-actual-api-key-here",
  authDomain: "your-project.firebaseapp.com",
  projectId: "your-actual-project-id",
  storageBucket: "your-project.appspot.com",
  messagingSenderId: "123456789",
  appId: "your-actual-app-id"
};

Step 5: Set Up Firestore Security Rules

In Firebase Console, go to Firestore Database ‚Üí Rules and update them:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Allow read access to commissions for everyone
    match /commissions/{document} {
      allow read: if true;
      allow write: if false; // Only allow writes through admin interface
    }
  }
}
Note: These rules allow public read access but no write access. You'll manage data through the admin interface only.

Step 6: Test Your Setup

  1. Save your changes to scripts/firebase-config.js
  2. Open your website and go to the admin panel (/admin.html)
  3. Log in with password: 3272
  4. Try adding a test commission entry
  5. Check if it appears on the commission queue page (/queue.html)

  • Using environment variables for sensitive config
  • Adding the firebase-config.js file to .gitignore if needed
